# Material de alumnos - Métodos Cuantitativos

Este repositorio reúne el material de apoyo del curso **Métodos Cuantitativos**.

Aquí encontrarás, organizado por semana:

- presentaciones en HTML
- presentaciones en PDF
- guías de estudio en PDF

## Organización del material

El contenido está ordenado en carpetas por semana:

- `semana 01/`
- `semana 02/`
- ...
- `semana 16/`

Dentro de cada carpeta encontrarás archivos con una numeración consistente, por ejemplo:

- `01_nombre_clase.html`
- `01_nombre_clase.pdf`
- `01_guia_estudio_nombre_clase.pdf`

## Cómo usar este material

- Si quieres revisar la clase de forma rápida, abre el archivo `HTML`.
- Si prefieres descargar o imprimir, usa el archivo `PDF`.
- Si necesitas repasar conceptos, ejemplos y ejercicios, revisa la `guía de estudio`.

## Recomendación

Se sugiere revisar el material en este orden:

1. Presentación de la clase
2. Guía de estudio
3. Ejercicios y repaso personal

## Importante

- El material está organizado siguiendo la secuencia oficial del curso.
- Los nombres de los archivos están estandarizados para facilitar su descarga y búsqueda.
- Si algún archivo no abre correctamente, vuelve a descargarlo desde el repositorio.

## Unidades del curso

- **Unidad I:** Estadística y Probabilidades con Python
- **Unidad II:** Test de Hipótesis y Regresión Lineal
- **Unidad III:** Regresión Logística
